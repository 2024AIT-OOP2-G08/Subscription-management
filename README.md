## 仕様

- flaskをサブスクリプション管理アプリ
  - 月毎の支払い合計を円グラフと値段降順データリストで表示する（ルート）
  - 円グラフから詳細リンクに飛び、詳細内容を表示する
  - データの登録、編集、削除をできるようにする
  - タグで管理する（余裕があれば）
